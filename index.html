HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Lang tag is the language of the page. EN for English
  • Head is the metadata or information data
  • Style is for CSS styles
  • Img is an image tag. src is the source url. Alt is the description. Width and Height set the image size

CSS

  • Style tags can be used to change the look of a item
  • Margin indicates how much space we want aroung the outside of the element
  • Padding indicates how much space we want on the inside of the element
  • header, footer means both the heading and the footer are applied to the tag
  • . before a class will apply it to all elements with the class

Git

  • Use git status to check which branch you're on
  • git checkout -b branch-name to create new branches and switch to it
  • git add -A: commits all changes
  • git commit -m "" sends a message about your changes
  • git push: push your changes to the site
  • git pull: recieve changes from the site
